Nhập từ file Excel

Cách dùng: Tải file mẫu về, điền từ vựng (cột: Chữ · Pinyin · Nghĩa · Buổi học · Câu ví dụ) rồi lưu ở định dạng CSV UTF-8. Sau đó bấm "Tải file lên".

Sao lưu & khôi phục dữ liệu

Dữ liệu hiện chỉ lưu trong trình duyệt này. Hãy Xuất ra file để sao lưu, đề phòng mất khi xoá cache/đổi máy. Khi nhập file, bạn có thể chọn thay thế hoặc gộp thêm.
